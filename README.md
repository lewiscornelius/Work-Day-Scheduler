# Work-Day-Scheduler

## User Story
```
AS A  busy professional
I WANT to easily schedule my daily tasks 
SO THAT I can manage my time effectively and efficiently
```

## Acceptance Criteria

```
GIVEN I am using the application at any point in time
WHEN I open the planner
THEN the current day is displayed at the top of the calendar
WHEN I scroll down
THEN I am  presented with time blocks for standard business hours: 9 a.m. - 5 p.m
When I view the timeblocks
THEN each timeblock is color coded to indicate whether it is in the past, present or future
WHEN I click into a timeblock
THEN the text for that event is saved in local storage
When I refresh the page
THEN the saved events persist
```
